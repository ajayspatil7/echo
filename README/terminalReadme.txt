ECHO --Electronic Code Held Object

Terminal Version-0.0.1

Terminal --Introduction

- A continuously running while loop carrying an input statement in INPUT_ variable.
- INPUT_ values are verified and sent to kernel.py to perform the operation.
- processor.py performs few more necessary operations which are not performed by terminal.py.
- cpu.py is a central unit where all the other part of program gets to process.

It is a 32-bit RISC-V CPU with a 32-bit wide data bus.
It is designed to be used in a space vehicle, and as such, It is designed to be very power efficient.
It is also designed to be very simple, and as such, It is not a general purpose CPU.
